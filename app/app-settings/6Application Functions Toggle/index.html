App Status

Allow users to open your app or not.

HWID Lock

Lock users to a value from your user's computer which only changes if they reinstall windows. Use this to prevent people from sharing your product.

Force HWID

Prevent users from logging in with a blank HWID (disable this for PHP)

VPN

Block IP addresses associated with VPNs

Hash Check

Checks whether the application has been modified since the last time you pressed the reset hash button. Used to prevent people from altering/bypassing your app.

Block Leaked Passwords

Prevent users from using leaked passwords when registering an account.

Token Validation

Checks to see if a user has a valid token to use your application.

Minimum HWID Length

Prevents users from logging in with a shorter HWID than the assigned value.

Version

The version of your application. Make sure to change it in your application as well.

Download Link

This is the link that will open if the version is different than the one in your application. (AKA auto-update).

Web Downloader

URL link for the web loader. (this will enable the web loader if it is not empty)

Webhook

Receive secure Discord webhooks for logs/activity.

Show IPs

Disable if you don't want IPs visible in Discord webhook logs

HWID Reset Cooldown (unit)

The unit before a user can HWID reset again.

HWID Reset Cooldown (duration)

The duration before a user can HWID reset again. (Unit * Duration = cooldown)

Session Expiry (unit)

The unit before the users session expires (logs out)

Session Expiry (duration)

The duration before the users session expires (logs out. Unit * Duration = expiry)

Minimum Username Length

Prevents users from creating an account with a username less than the given value.

It's highly recommended to change these to something custom!
